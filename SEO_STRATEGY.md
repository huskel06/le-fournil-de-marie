## Structure de maillage interne

```
Accueil (/)
├── Savoir-Faire (/savoir-faire) — lien "Découvrir notre savoir-faire"
├── Contact (/contact) — lien "Nous écrire"
├── À Propos (/a-propos) — lien dans footer
└── Commande (/commande) — lien dans footer

Savoir-Faire (/savoir-faire)
├── Contact (/contact) — CTA "Nous contacter"
└── Accueil (/) — breadcrumb

À Propos (/a-propos)
├── Contact (/contact) — navigation
└── Accueil (/) — breadcrumb

Contact (/contact)
├── Accueil (/) — breadcrumb
└── Commande (/commande) — mention dans chapô

Commande (/commande)
├── Contact (/contact) — fallback email/tel
└── Accueil (/) — breadcrumb
```

## Recommandations contenu

1. **Page Accueil** : H1 "Le Fournil de Marie" + sous-titre optimisé avec "boulangerie bio Nantes Centre"
2. **Page Savoir-Faire** : Contenu riche sur le levain naturel, mots-clés "pain au levain Nantes", "farines bio Vendée"
3. **Page À Propos** : Mettre en avant "seule boulangerie bio du quartier" + "farines locales moins de 100 km"
4. **Page Contact** : Adresse complète avec code postal 44000, horaires, plan
5. **Page Commande** : Cibler "mariage Nantes", "pièce montée Nantes", "buffet viennoiseries"
